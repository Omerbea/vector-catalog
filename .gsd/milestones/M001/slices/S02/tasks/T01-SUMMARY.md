# T01: Project scaffold — pom.xml, entity, repository, service, controller

**Full Spring Boot project compiled and running: POST /api/products returns 201 with embedding auto-generated by DB trigger; GET /api/products/{id} returns 200/404.**

## What Happened

Built all 9 source files. Two fixes required during integration:

1. `price` as `Double` with `@Column(scale=2)` — Hibernate rejects scale on floating-point types. Fixed to `BigDecimal`.
2. `SERIAL` in init.sql — Hibernate `Long id` expects `bigint`. Fixed to `BIGSERIAL`.

After fixes: `mvn compile` succeeded, `mvn package` produced 51MB fat jar, Spring Boot started in 2.4s, both endpoints tested successfully via curl. DB trigger confirmed firing — `embedding IS NOT NULL AND vector_dims(embedding) = 768` after every POST.

## Deviations

- `price`: `Double` → `BigDecimal` (Hibernate constraint)
- `id` sequence: `SERIAL` → `BIGSERIAL` in init.sql (Hibernate type validation)
